Project Requirements:

HR department requires the ability to do the following items:   
    ●   Ability to view all team members on a single page. - ACCOUNTED
    ●   Add a new team member as they are hired, delete a team member record when needed, and mark a team member as terminated. - ACCOUNTED
    ●   Edit a team member’s record, i.e. address, position, department, etc. -ACCOUNTED
    ●   An activity tracker that will record when a team member’s permissions change, when a team member’s manager changes, 
    and when a team member’s position changes. - ACCOUNTED
    ●   A reporting page to view recruiting, and retention efforts including: - ACCOUNTED
           ○    Weekly hire numbers
           ○    Terminated numbers for the current year
    ●   In an effort to maintain the site’s security have a page to view all permission assignments for all team members. - ACCOUNTED

HR has asked that the following items be included in the team member record:
    ●   Name
    ●   Address
    ●   Email Address
    ●   Preferred Contact Phone Number
    ●   Position
    ●   Department
    ●   Start Date
    ●   End Date
    ●   Employment Status
    ●   Shift
    ●   Manager
    ●   Team Member Photo
    ●   Favorite Color

For ALL options, the application should include the following:
    ●   No authentication required
    ●   Data should be stored in a SQL Database (preferred but not required to be in SQL database)
    ●   A way to view all team members in the database   -   ACCOUNTED
    ●   Ability sort and filter employees on the all team page -  ACCOUNTED
    ●   Ability to search for team members by name on the all team page -  ACCOUNTED
    ●   The ability to add, delete, and edit team member records  -   ACCOUNTED
    ●   A reporting page for displaying recruiting and retention metrics - ACCOUNTED
           ○    Weekly hire numbers
           ○    Terminated numbers for the current year
    ●   A page to view the activity log of modifications made to a team member record - ACCOUNTED
   ●  Page to view permission assignments. ACCOUNTED

Nice Features, to include after you complete the MVP:
    ●   Photo upload for team member photos - ACCOUNTED
    ●   Ability to create new departments/positions - ACCOUNTED
    ●   A report view detailing the number of employees in each department/manager - ACCOUNTED
    ●   A tree hierarchy view of the management chain throughout all employees. - ACCOUNTED
    ●   A notification system that alerts certain users when changes are made within the system. - ACCOUNTED
